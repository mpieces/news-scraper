## About The Project
(Under Development -- WORKS LOCALLY)
News Scraper is a web scraper that scrapes Buzzfeed News for the latest news articles.

* CSS
* [Bootstrap](https://getbootstrap.com/)
* JavaScript
* [jQuery](https://jquery.com/)
